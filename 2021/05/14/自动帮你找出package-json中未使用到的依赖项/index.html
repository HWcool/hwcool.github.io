自动帮你找出package.json中未使用到的依赖项

depcheck

一个中大型的项目通常会有很多的包依赖项，在维护这些项目时，如果想对项目做一些优化的动作时，对包依赖分析是一个必不可少的动作。
但是对于上百个依赖项中手动查找出哪些用过，哪些没用过的就很头疼

下边这个工具只要执行一行代码，就能自动帮你分析出哪些是没有用过的包。
