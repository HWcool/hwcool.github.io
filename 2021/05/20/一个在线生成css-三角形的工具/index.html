一个在线生成css 三角形的工具

css-triangle-generate

在线预览地址

实现了如下的基本功能

  • 可调整三角形的高度
  • 可调整三角形的宽度
  • 可调整三角形的角度
  • 可调整三角形的背景色

使用了Vue3.0 版本 和ant-design 写的一个小玩意，具体也没啥好说的，比较简单功能的一个页面。主要是想体验下vue3.0版本的使用方法…

自vue3.0出来之后，一直没有用过，一部分是因为当前工作中事情比较多，另一方面是因为当前工作中使用vue2.0版本比较多的。这次就拿这个来试试手吧

也算对vue3的语法有个基本了解了吧
